# 291-Mini-Project-I
A command line application for ride sharing using SQLite as our database for CMPUT 291 Mini-Project-I

## Instructions
To Install:
```
./install.sh
```

To run:
```
python3 main.py prj.db
```

To run with your own database file:
```
python3 main.py <path_to_database>
```
